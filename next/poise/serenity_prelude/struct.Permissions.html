Struct Permissions

source 
#[repr(packed(1))]
pub struct Permissions(/* private fields */);
Expand description

A set of permissions that can be assigned to Users and Roles via PermissionOverwrites, roles globally in a Guild, and to GuildChannels.

Discord docs.
