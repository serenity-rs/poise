pub struct PrivatePkcs1KeyDer<'a>(/* private fields */);
Expand description

A DER-encoded plaintext RSA private key; as specified in PKCS#1/RFC 3447

RSA private keys are identified in PEM context as RSA PRIVATE KEY and when stored in a file usually use a .pem or .key extension. For more on PEM files, refer to the crate documentation.
