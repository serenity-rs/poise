A macro which returns the result of polling a future once within the current async context.

This macro is only usable inside of async functions, closures, and blocks. It is also gated behind the async-await feature of this library, which is activated by default.

If you need the result of polling a Stream, you can use this macro with the next method: poll!(stream.next()).
