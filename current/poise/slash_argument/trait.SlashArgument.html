source

fn create(builder: CreateCommandOption) -> CreateCommandOption

Create a slash command parameter equivalent to type T.

Only fields about the argument type are filled in. The caller is still responsible for filling in name(), description(), and possibly required() or other fields.

Don’t call this method directly! Use crate::create_slash_argument!
