Expand description

Given a Message and some context data, parses prefix, command etc. out of the message and returns the resulting crate::PrefixContext. To run the command, see run_invocation.

Returns Ok(None) if the message does not look like a command invocation. Returns Err(...) if the message does look like a command invocation, but cannot be fully parsed.
